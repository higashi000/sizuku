# sizuku

## Description
TUI task magement tool.

## Library
- [tview](https://github.com/rivo/tview)

## Usage
- add task
```
$ sizuku -a

Task Title >> YOUR TASK TITLE
Task Details >> YOUR TASK DETAILS
Task Limit >> YOUR TASK LIMIT( default none )
```

- see task
```
$ sizuku -s
```

- delete task
```
$ sizuku -d
```
